1. Project Goals
----------------
- Create a completely custom system-wide 12-band equalizer as a Windows Audio Processing Object (APO).
- Fully compatible with FL Studio 2025 (and any other DAW or application using WASAPI).
- Works with Steinberg UR22mkII (when set as default output device).
- Sample rate: 44.1 kHz primary, with fallback support for other rates.
- 12 sliders (one per band) + 1 Master Gain slider.
- Special gain behavior: В±10 dB with 1 dB steps; outside this range в†’ 0.1 dB precision.
- NY-Style parallel compression + soft saturation (activates when Master Gain > 0 dB).
- Switchable Dark theme в†” Neon theme.
- Small resizable window + System Tray support.
- Pure output processing only (no input/recording path).
- Designed for learning OS-level audio development.
